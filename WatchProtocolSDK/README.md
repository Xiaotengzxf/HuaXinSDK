# WatchProtocolSDK

智能手表蓝牙通信协议 SDK for iOS

## 版本历史

- [v1.0.1](./v1.0.1/) - 2026-01-03
  - 首次发布
  - 完整的蓝牙设备管理功能
  - 支持步数、睡眠、心率、血氧、血压数据同步
  - 协议化数据存储设计

## 快速开始

### 集成 SDK

1. 下载对应版本的 `WatchProtocolSDK.xcframework`
2. 将 framework 拖入 Xcode 项目
3. 设置 Embed 为 "Embed & Sign"
4. 添加蓝牙权限到 Info.plist

详细说明请参考完整文档。

## 系统要求

- iOS 13.0+
- Xcode 12.0+
- Swift 5.0+
