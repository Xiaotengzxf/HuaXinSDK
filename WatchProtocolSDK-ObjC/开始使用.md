## 🎯 快速集成（3 步）

### 第 1 步：拖入 Framework

将 `WatchProtocolSDK.xcframework` 拖入你的 Xcode 项目

### 第 2 步：配置 Embed

在 Target → General → Frameworks, Libraries, and Embedded Content：
- 设置为 **Embed & Sign** ⚠️

### 第 3 步：添加代码

```objc
// AppDelegate.m
#import <WatchProtocolSDK/WatchProtocolSDK.h>

- (BOOL)application:(UIApplication *)application
    didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {

    // 初始化 SDK
    [[WPBluetoothManager sharedInstance] initCentral];

    return YES;
}
```

✅ **完成！** 现在可以开始扫描和连接蓝牙设备了。

## ❓ 遇到问题？

### 常见问题

1. **找不到头文件？**
   → 确保 Embed 设置为 "Embed & Sign"

2. **运行时崩溃？**
   → 检查是否添加了 CoreBluetooth.framework

3. **蓝牙无法使用？**
   → 检查 Info.plist 中的蓝牙权限
